# aus_data_jobs

## pre-requisites

### install PostgreSQL locally on machine
### create an account with Adzuna jobs api to obtain API keys
### create .env file mapped values to:

#### .env example

postgres creds
DB_USER=<xx>
DB_PASSWORD=<xx>
DB_HOST=<xx>
DB_PORT=<xx>
DB_NAME=<xx>

adzuna api creds
API_APP_ID=<xx>
API_APP_KEY=<xx>
